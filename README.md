# HealthCare-ML-corusework
Machine Learning on Healthcare datasets(Pims diabetes)
- **Dataset:** Pima Indians Diabetes Dataset
- **Goal:** Predict whether a patient has diabetes or not
- **Models Used:**  Logistic Regression
- **Libraries:** Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
- **Skills Demonstrated:** Data Cleaning, Feature Engineering, Model Training and Evaluation
